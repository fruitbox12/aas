# TopBar (Dashboard)

<br />

## Uses

Serves as the navigation bar on the dasboard. <br />

## Children Components

Imports `Account` from `src/layouts/DashboardLayout/TopBar/Account.js`.
Imports `Settings` from `src/layouts/DashboardLayout/TopBar/Settings.js`.
Imports `Logo` from `src/components/Logo.js`.

## Implementation

### Props Required

```JavaScript
props = {
   className,
   onMobileNavOpen
}
```

### Symbols to Note

| Variable          | Type         | Purpose                                                  |
| ----------------- | ------------ | -------------------------------------------------------- |
| `className`       | string       | Custom className for sizing and positioning              |
| `onMobileNavOpen` | function     | Function that controls if the `NavBar` appears on mobile |
| `classes`         | object       | ClassName generated by MUI for styling                   |
| `navLinks`        | object array | Array of nav links to render in the `TopBar`             |

## Expected Behavior

On desktop, used as the navigation bar.

-  Has the links to the other pages
-  Dispalys the logo
-  Shows the settings
-  Shows the account

On mobile, navigation is in the `NavBar` accessible through the hamburger menu.
