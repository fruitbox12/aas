# Footer

<br />

## Uses

On the bottom of the home page containing useful links to the community and developer resources.

## Implementation

Mapping the`SOCIAL` constant to map out the links.

## Expected Behavior

On a horizontally large viewport, the components are spread out horizontally.
Otherwise, the two halves become vertically stacked.
