# Name Card

<br />

## Uses

Used to show some title with a splash of color. Specifically being used for the oracle profile page to say the name of the provider associated with the page.

## Implementation

### Props Required

```JavaScript
props = {
   className,
   name,
}
```

### Symbols to Note

| Variable    | Type   | Purpose                                     |
| ----------- | ------ | ------------------------------------------- |
| `className` | string | Custom className for sizing and positioning |
| `name`      | string | The name/title to dispaly                   |

## Expected Behavior

View is fluid. Restriced only by being palced within a container in a HOC.<br />
The primary color of the palette should be visible as a gradient on the right hand side of the card.
